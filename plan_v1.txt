WYSHMATE — PHASE 0 MVP BUILD DOCUMENT
🧠 Product Name
Wyshmate

🎯 Objective
Build a minimal web app where users create a celebration board, share a link, and others add text wishes.

❗ Scope
✅ Included
Create board

Share link

Add text message

View messages

❌ Not Included
Authentication

Video/audio

AI

Payments

Notifications

🧩 User Flows
Create Board
Click “Create Board”

Enter occasion, recipient name, optional title

Generate public + admin link

Redirect to board

Share
Copy public link

Send to others

Add Message
Enter name + message

Submit

Message appears
